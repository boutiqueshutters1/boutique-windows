Double Hung Windows

Classic American-style windows with both sashes operable. Superior security with ANSI/BHMA Grade 1 locking and exceptional noise reduction up to 35dB.

Home › Windows › Double Hung Windows

120T Series

The 120T Double Hung features a substantial 122mm profile depth with 1.6mm wall thickness. ANSI/BHMA Grade 1 locking system deters break-ins, exceeding California Title 24 requirements. Argon-filled cavities and warm-edge spacers minimize condensation in humid regions. AAMA 2605-certified aluminum frame resists salt corrosion in coastal states. 35dB sound insulation (ANSI S12.60) ideal for urban areas near airports or highways.

Profile Thickness 1.6mm
Profile Depth 122mm
Standard Glass 5mm Low-E + 9Argon + 5mm Tempered Clear Glass
Standard Hardware KWI International
